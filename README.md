# VotaAppAPI

Backend to serve VotaApp

### Getting Started

Install Node.js (https://nodejs.org/en/)

Run command 'npm install' at project's folder.

Import votaApp.sql in a MySQL database

db config: {
        'name': 'votaapp',
        'user': 'root',
        'password': 'admin'
        'hostdb': 'localhost'
}

server runs on port 7780
